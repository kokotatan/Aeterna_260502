# Aeterna プロジェクト概要資料（NotebookLM 読み込み用）

## 1. プロジェクトの基本情報
*   **プロジェクト名**: Aeterna（エターナ）
*   **キャッチコピー**: 「あなたの想いを、永遠に動き続けるAI分身に託す。終活特化型・意思を残す遺産分配ソフトウェア」
*   **目的・コンセプト**: 
    ユーザーが生前に持っていた価値観、哲学、口癖、そして「どんな世界になってほしいか」という想いをAI（分身）に学習させます。ユーザーの死後も、そのAI分身が24時間365日、世界のニュースや情勢を監視し続けます。そして、生前の価値観（トリガー条件）に合致する重大な出来事（気候変動、紛争、教育問題など）が発生した際に、自律的に判断し、適切な支援団体や個人に対して暗号資産（遺産）を自動的に寄付・送金します。

## 2. コア体験（ユーザーフロー）
1. **人格の抽出（Discovery）**
    *   ユーザーは普段使い慣れた LINE アプリを通じて、Aeterna の「記憶の守り手（AI）」と対話します。
    *   自然な会話の中で、ユーザーの誇り、人生哲学、関心事、次世代へ残したい想いをAIが引き出し、最終的に「AI分身（Twin Persona）」のプロファイル（JSONデータ）を合成します。
2. **人間証明とエージェントの公式登録（World ID & AgentKit）**
    *   金融資産を扱うAIエージェントがスパムや不正利用されないよう、Worldcoin の **AgentKit** と **World ID** を用いて「このエージェントは確かに実在の人間によってバックアップされている」ことをブロックチェーン上（World Chain / Base）に証明・登録します。
    *   ユーザーは LINE に送られてくるリンクから World App を立ち上げ、QRコードをスキャンするだけで承認が完了します。
3. **自律的な監視と遺産分配（Execution）**
    *   AI分身は設定されたトリガー（例：「海面上昇の危機」「教育格差の拡大」）を元に、世界のニュースを監視します。
    *   条件を満たすニュースを検知すると、Anthropic (Claude) の推論エンジンが「故人ならどう考えるか」を判断し、ブロックチェーンを通じて自動で寄付（送金）を実行します。
    *   また、LINEから直接「孫の〇〇へ10 USDC送金して」と指示を出すことで、AIが自律的にツールを呼び出して送金や換金を行うインタラクティブな操作も可能です。

## 3. 技術スタック（Tech Stack）
*   **言語**: JavaScript (Node.js), HTML, CSS
*   **バックエンドフレームワーク**: Express
*   **AI / 推論エンジン**: Anthropic API (Claude 3.5 Haiku) / OpenClaw アーキテクチャ
*   **ユーザーインターフェース**: LINE Messaging API (LINE Bot)
*   **認証・アイデンティティ**: World ID, AgentKit (@worldcoin/agentkit)
*   **ブロックチェーン・金融レイヤー**:
    *   **Base (EVM)**: ethers.js を使用したスマートコントラクト操作。USDC（ステーブルコイン）の直接送金や、Uniswap V3 を経由した ETH から USDC への自動換金（スワップ）機能。
    *   **XRP Ledger (XRPL)**: 高速かつ低コストなグローバル寄付送金（xrpl.js）。

## 4. プロジェクトの革新性・ハイライト
1. **シニア層に寄り添ったUX**: 専用アプリや難解なWeb3ウォレットの操作を極力排除し、日常的に使う LINE のチャットボットだけでAIとの対話や送金指示が完結する。
2. **AI × Crypto の融合**: Anthropic の Tool Calling 機能（関数呼び出し）を利用し、AI自身が「今、換金すべきか」「誰に送金すべきか」を考え、自律的にスマートコントラクトを叩く（`transferUSDC` や `swapETHToUSDC` の実行）。
3. **World ID によるエージェントの信頼性担保**: AIエージェントが自律的に経済活動を行う未来において懸念される「ボットによるスパム攻撃」を、AgentKit と World ID の生体認証プロセスを組み込むことで解決。エージェントが「人間に裏付けられた安全な存在」であることをオンチェーンで証明している。
    *   *※ハッカソン中の成果として、この仕組みを用いて「人間認証済みエージェント限定」の Shopify ストア制限を突破し、専用商品（Human in the loop Hat）の決済を成功させました。*

## 5. デモ用シナリオ例
*   **シナリオ1（人格作成と認証）**:
    LINEでボットに「始める」と送信。AIと会話して「環境保護と教育を重んじるKotaro」の人格が合成される。その後、World Appでエージェントを承認する。
*   **シナリオ2（手動での資金操作）**:
    LINEから「手持ちの0.001 ETHをUSDCに換金（スワップ）しておいて！」と送信。AIがUniswapのコントラクトを叩き換金を実行し、完了通知とTXハッシュを返す。
*   **シナリオ3（自律的な遺志の執行）**:
    LINEから「デモ実行」と送信。AIが最新の気候変動ニュースを読み取り、「故人の遺志（環境保護）に合致する重大な事態である」と判断。WWFジャパン等のウォレットへ、設定された割合の暗号資産を自動で送金・寄付する。
